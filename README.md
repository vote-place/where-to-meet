## Tech Stack

### Frontend
- Next.js 16
- React 19
- TypeScript

### Styling
- Tailwind CSS 4

### Backend
- Next.js Route Handlers (API)

### Database
- MySQL (mysql2)

### Map / External API
- @vis.gl/react-google-maps

### Code Quality
- ESLint
